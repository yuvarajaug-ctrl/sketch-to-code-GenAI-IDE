# 🎨 Sketch to Code - GenAI IDE

An AI-powered application that converts hand-drawn sketches into functional UI code.

## 📌 Project Overview
This project uses Machine Learning and Computer Vision techniques to detect UI elements from sketches and automatically generate code.

It helps developers and designers quickly transform ideas into working applications.

---

## 🛠️ Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning (AI/ML models)
- OpenCV (for image processing)

---

## ✨ Features
- Upload hand-drawn UI sketches
- Detect UI components automatically
- Convert sketch into structured code
- Simple and user-friendly interface
- Faster prototyping for developers

---

## 📂 Project Structure
- `models/` → ML models
- `routes/` → Backend routes
- `templates/` → Frontend UI
- `static/` → CSS, JS, uploads
- `database/` → Data handling

---

## Run
1. `pip install -r requirements.txt`
2. `mysql -u root -p < database/schema.sql`
3. `python app.py`

Go to `http://localhost:8000/`
